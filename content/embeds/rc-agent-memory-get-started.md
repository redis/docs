To set up Agent Memory on Redis Cloud:

1. [Create a database]({{< relref "/operate/rc/databases/create-database" >}}) on Redis Cloud.
2. [Create an Agent Memory service]({{< relref "/operate/rc/agent-memory/create-service" >}}) for your database on Redis Cloud.
3. [Use the Agent Memory API]({{< relref "/operate/rc/agent-memory/use-agent-memory" >}}) from your client app.

After you set up Agent Memory, you can [view and manage your service]({{< relref "/operate/rc/agent-memory/view-service" >}}).
