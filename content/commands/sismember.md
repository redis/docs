Returns if `member` is a member of the set stored at `key`.

## Examples

{{% redis-cli %}}
SADD myset "one"
SISMEMBER myset "one"
SISMEMBER myset "two"
{{% /redis-cli %}}

* [Integer reply](../../develop/reference/protocol-spec#integers): `0` if the element is not a member of the set, or when the key does not exist.
