## Get health report {#get-crdbs-health}

	GET /v1/crdbs/{crdb_guid}/health_report

Get a health report for an Active-Active database.

#### Example HTTP request

#### URL parameters

| Field | Type | Description |
|-------|------|-------------|

| Field | Type | Description |
